勝ち手Otto4 Othello Ver 6

1. 手番の設定
　[setting]でcomputer の手番を決めます
　１手目は対称性から[4,3]に強制します
  humanは　マウス　click　ですが　[computer]　clickでcomputerに打たせます。
　「待った」は[undo]ボタンでできます。



2. computerの戦略
　序盤（17手まで）は定石サーチ
　終番(48手以降) はαβ法で最善手サーチ
　準終番(33-47)は　df-pnf法で勝ち手を探します
　勝ち手がないときは　コミを増やして繰り返します
　中盤は評価関数の７手先読みで最善手を探します

3　高段者の局面
　　program No と　step No で指定した局面に　lopadpositiomで　
　　行くことができます　　「kiｆ」の　<< 　<　>>　で巡回できます。
　　[computer]ボタンで　computerに打たせることができます。

4, df-pn法
　　「深さ優先　証明数検索」
　　DF-PN　Search =　Depth First Proof Number　Search
　　詰将棋で威力を発揮している。
　　詰将棋では若いnodeで　枝刈りできるのに対して　オセロでは
　　60　手まで進まないと　勝ち負けがわからない。
　　33手目が限界になる。（読みの深さ 27）

　　そこで　今４隅が空きとしたら、先に隅に打てば勝ち、
　　また１隅が打たれているなら　次に隅を打っ方が勝ちとする。
　　この　cornerwin optionで　30手の勝ちが見つかる

5. サーチフェーズの打ち切り時間
　　指定によって　30,60,100 ,200 秒に変えられる
　　defualtは　60秒
